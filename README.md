# She out | online store

A very simple e-commerce website , that the seller can use by editing on the code , and the costumer can use by clicking the website link

[Live Demo](https://retagaboshady.github.io/selling-web/)

## Features

* **Search Bar:** It is responsible for helping you fing items if you are searching for an specific item 

* **Category:** It is responsible for helping you in shopping when you dont know  what you really want you just choose the category (Steetwear-Beauty & Makeup-Accessories)

* **Cart:** It shows the total cost of your order and your whole order

## Tech Stack

* **HTML5:** Structing the web page

* **CSS3:** Styling,layout (Flexbox/Grid) , and custom animation

* **JavaScript(ES6+):** Handlling DOM manipalation , logic , and interactivity

* **JSON:** Mangaing and storing data stracture.
